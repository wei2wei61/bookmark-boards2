使い方ガイド

カードを追加する

上部の入力欄に X（旧Twitter）または YouTube の URL を貼り付けて「保存」を押すと、そのポスト・動画がカードとしてボードに追加されます。Enter キーでも保存できます。同じ URL はすでに保存済みと判定され重複追加されません。

テキストメモを残したい場合は「メモ」ボタンから自由記述カードを作成できます。

タグ・検索・絞り込み

カードにホバーすると下部にメモ欄とタグ入力欄が開きます。タグ名を入力して Enter キーで追加。以前付けたタグはワンクリックで再利用できます。

ヘッダー右の タグアイコン を押すとタグフィルターが展開されます。タグを複数選択すると AND 条件（すべてのタグを持つカードだけ）で絞り込めます。選択解除は「✕ クリア」ボタンで一括リセットできます。

ヘッダー右の 虫眼鏡アイコン をクリックすると検索バーが展開します。メモ・メモカードのタイトルや本文をリアルタイムで絞り込みます。検索バーからマウスを外すと自動的に閉じます。

ボードとフォルダで分類する

サイドバーの 各アイコン からボードやフォルダを作成できます。ボード名の横にある鉛筆アイコンで名前の変更、書き出しアイコンでそのボードだけを JSON ファイルとして出力できます。

ボードをフォルダ内にドラッグ＆ドロップするとフォルダへ移動できます。フォルダをクリックすると折りたたみ／展開できます。

ドラッグ＆ドロップで並び替え・複製

カードをつかんで別のカードの上にドロップすると 順番を入れ替え できます。サイドバーの別のボードへドロップすると、そのカードを 複製（コピー） してボードへ追加します（元のボードからは削除されません）。

データの保存とバックアップ

● Canvas 環境（claude.ai）：編集内容は自動的にクラウドへ保存されます。

● ブラウザ単体で使う場合：データはブラウザのローカルストレージに保存されます。ブラウザのデータを消去するとデータも消えます。サイドバーの データ出力アイコン から定期的に JSON バックアップを取っておくことを強くおすすめします。バックアップはデータ取込アイコンから復元できます。
