使い方ガイド

カードを追加する

上部の入力欄に X または YouTube の URL を貼り付けて「保存」を押すと、現在のボードにカードを追加できます。Enter キーでも保存できます。同じ URL は重複して追加されません。

YouTube はサムネイルをクリックするとカード内で再生できます。右上の全画面アイコンから大きく表示でき、外部で開きたい場合はカード下部のリンクを使えます。

自由に書き残したい内容は「メモ」ボタンからテキストカードとして追加できます。

タグ・検索・絞り込み

X のカードには x、YouTube のカードには youtube タグが自動で付きます。カード下部のタグ入力欄にタグ名を入力して Enter キーを押すと、任意のタグも追加できます。

タグアイコンを押すとタグフィルターが開きます。複数タグを選ぶと、すべてのタグを持つカードだけを表示します。選択を戻すときは「クリア」を押します。

検索アイコンを押すと検索バーが開きます。メモ、テキストカードのタイトル、本文をリアルタイムで絞り込めます。

ボードとフォルダで分類する

サイドバーからボードやフォルダを作成できます。ボード名やフォルダ名は鉛筆アイコンで変更でき、不要になったものは削除できます。

ボードをフォルダへドラッグすると分類できます。フォルダ名をクリックすると、ボード一覧を折りたたみ／展開できます。

ドラッグ＆ドロップで並び替え・複製

カードをドラッグして別のカードの位置へ動かすと並び替えできます。サイドバーの別ボードへドロップすると、そのカードをコピーして追加します。元のボードのカードは残ります。

データの保存とバックアップ

データはブラウザのローカルストレージに保存されます。ブラウザのデータを消去すると保存内容も消えるため、サイドバーのデータ出力アイコンから定期的に JSON バックアップを取ってください。

バックアップファイルはデータ取込アイコンから復元できます。ボードごとの書き出しもできるため、テーマ別に共有・保管したいときに使えます。
